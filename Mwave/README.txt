First, make file named "Mwave" in "bin" folder executable.

You need to install following dependencies before you can start using Mwave properly:
Python dependencies:

Install python installer which will help you install other python packages directly: sudo apt-get install python-pip/

1. pygsr 0.1
	Run : pip install pygsr
	Or,
	Get it from this URL : https://pypi.python.org/pypi/pygsr/0.1

	This depends on:
	a) sox 
		Run: sudo apt-get install sox
	b) pyaudio 
		Run: sudo apt-get install python-pyaudio

2. feedparser 5.1.3
	Run : pip install feedparser
	Or,
	Get it from here : https://pypi.python.org/pypi/feedparser

3. python-weather-api(pywapi)
	Run : pip install pywapi
	Or,
	Get it here : https://code.google.com/p/python-weather-api/


Other dependencies(You can also install these from Ubuntu Software Center, if you system is Ubuntu):

1. espeak
	Try to install it from Ubuntu Software Center, or else
	Get it here : http://espeak.sourceforge.net/download.html

2. xte(part of xautomation)
	Run: sudo apt-get install xautomation

3. xset(by default part of Ubuntu 13.10)

4. amixer(by default part of Ubuntu 13.10)

5. xrandr(by default part of Ubuntu 13.10)
